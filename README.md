
# ToDo DApp

## **Decentralized ToDo WebApp**

 - [x] *Final Branch -* ```fin```

---
## Requirements
- NodeJS : Node JS website
- Personal Blockchain : [Ganache](https://www.trufflesuite.com/ganache)
- Framework : Truffle
- Extension : Metamask

---
## Steps to install

```
$ npm install
$ truffle compile
$ truffle migrate
$ npm run dev
```
